# 🚀 FastAPI CRUD API

This is a simple CRUD (Create, Read, Update, Delete) API built using FastAPI and Python.

## 📌 Features
- Create new records
- Read all or single record
- Update existing data
- Delete records

## 🛠️ Tech Stack
- Python
- FastAPI
- Uvicorn

## ⚙️ Installation

```bash
git clone https://github.com/your-username/fastapi-crud-api.git
cd fastapi-crud-api
pip install -r requirements.txt
uvicorn app.main:app --reload (run the code)
